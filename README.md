# INX Employee Performance Analysis

This project analyzes employee performance data for INX Future Inc using data science
and machine learning techniques. The objective is to identify key factors affecting
performance, evaluate departmental trends, and build predictive models for HR decision-making.

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Jupyter Notebook
